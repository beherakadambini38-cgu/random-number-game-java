#  Number Guessing Game (Java)

##  Project Overview
This is a simple Number Guessing Game built using Java.  
The computer generates a random number, and the user has to guess it with hints like "Too High" or "Too Low".

This project helps in understanding loops, conditionals, and basic logic building in Java.

---

##  Features
- Random number generation (1–100)
- User input guessing system
- Hints: Too High / Too Low
- Tracks number of attempts
- Displays result when guessed correctly

---

##  Technologies Used
- Java
- Scanner Class
- Random Class
- Loops & Conditional Statements
